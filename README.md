# Michael-Dane-Lewis-Portfolio

Personal portfolio site for **Michael Dane Lewis** — Operations & Project Management professional.

Built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Production build
- `npm run start` — Start production server
- `npm run lint` — Run ESLint

## Deploy

This project can be deployed on [Vercel](https://vercel.com) or any platform that supports Next.js.
